THEME 02

Basic Content

In Theme 02 we learned mostly about The basic principles behind composing a picture frame for moving images and how different content types fits into an online context.

We worked with video and how to develop it in iterative steps using treatment and storyboard to professionalise the production.

We produced 2 short videos. One on mobile, where we Created a story without using spoken language. And the other one a bit more professional using actual video cameras where the goal was creating an informative video for an external collaboration partner.

02.01.02 XD prototype

In this task we were introduced to Adobe XD. We made an XD prototype of our own responsive "information" website which would include an information video, information about the group members and creators of the website, as well as some extra information related to the external collaboration partner we were given. By the end of this theme the skeleton of the website would be mostly completed.

02.01.03 Link to Video: Story without words

Here I had to create a story without a spoken language. It was challenging but very rewarding once I completed the task. I learned how to use Adobe Premiere Pro for the first time which improved my Video cutting skills greatly, and I have used it alot in my own free time since.
