THEME 03

Basic Animation

In Theme 03 We worked a lot with Adobe Illustrator and Javascript. This Theme was my favourite but also, by far, the most challenging in my opinion.

We worked a lot in groups but we also had some individual assignements. I can honestly say that I surprised myself a lot, going from such little knowlidge of Javascript into making an actual animation with it.

03.01.02. Link to simple animation

In this Task we were asked to make a character (eather created from scratch or something from the internet) do all kinds of stuff by using Javascript. I found myself a small animated Robot and by using Javascript I was able to make it jump around, lie down, Jump in circles, glow and all kinds of animation. It was very challenging and very frustrating if I was doing it wrong without knowing what it was that was wrong. But once I was able to make it work it was EXTREMELY REWARDING.

03.03.02 Interactive Animation

In these 2 tasks we were introduced to Storytelling and Character design.

We had to create our own stroy as well as our own characters. This was very fun to do. I learned a lot about creating and designing something from scratch. For example, creating a character you have to know his back story. What makes him look and act like he does.

Not only did we have to create characters but we had to aim the creation to a theme. "What kind of theme of animation are you using." I used flat design which is a bit more 2D-ish and what characterizes that theme is that is has no outside lines for example.

The second task was creating an animated part of the story by using HTML, CSS and JAVASCRIPT. This was a bit challenging. Making a sprite from the character to make him walk realisticly and then implementing it into the animation was very challenging, but really cool once it worked!
